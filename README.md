## 🌐 Next-Gen Portfolio Website (Full-Stack)

Welcome to the repository for my newly rebuilt **Full-Stack Portfolio**! This is a modern, interactive, and highly scalable web application designed to showcase my projects and skills, complete with a functional backend for real-world interactions (like bug reports and feedback).

### 🏗️ Tech Stack

- **Frontend:** React.js, Vite, Tailwind CSS, Framer Motion
- **Backend:** Python, FastAPI, SQLite (SQLAlchemy ORM)
- **Deployment:** Vercel (Frontend) & Render (Backend)

*The legacy static HTML version of this portfolio has been safely archived in the `old-html/` directory.*

---

## 🚀 Key Features

- **Interactive UI:** Futuristic design with glassmorphism, dynamic typing, and scroll-triggered animations.
- **API-Driven Project Cards:** Projects are fetched dynamically from the FastAPI backend.
- **Live Bug Reporting System:** An interactive modal allowing users to report project-specific bugs directly to the backend database.
- **Feedback Collection System:** Submit structured feedback with a sleek 5-star rating UI.
- **Smooth Navigation:** Sticky navbar, custom scrollbar, and fully responsive layout (mobile-first).
